🔷 5. Effects & Animations
CSS Animations vs JS Animations
CSS Keyframes (e.g. @keyframes heartGlowPulse): Used primarily for 2D container elements like buttons and glowing auras because the browser's Rendering Engine handles CSS opacity and transforms incredibly efficiently.
JavaScript Animations (e.g. Framer Motion <motion.div>): Used when components mount/unmount. Framer Motion intercepts the render, injects an inline CSS style, interpolates values, and then allows the component to fully delete itself from the DOM when finished.
WebGL Interactions (e.g. Three.js useFrame): Used for the Heart and Particles. Modifies raw GPU shader data instead of HTML DOM nodes.
DOM Manipulation Strategy
To prevent layout jumps, the application heavily utilizes CSS position: absolute and position: fixed. When the Special Message opens, it uses React's createPortal, injecting the cinematic sequence directly onto document.body instead of burying it in nested <div>s.
